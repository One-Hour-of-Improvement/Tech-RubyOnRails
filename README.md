# Ruby on Rails

Ruby on Rails is a web application framework that includes everything needed to create database-backed web applications according to the Model-View-Controller (MVC) pattern. Ruby on rails also has 2 main principles: DRY (Don't Repeat Yourself) and CoC (Convention Over Configuration).
